# Yandex Speechkit Bot
Бот поможет найти ответ на нужный вопрос и расшифровать текст в войсе или сделать голосовое сообщение из заданного текста
<br>
Доступные команды:
- /start - перезапуск бота
- /stt - войс в текст
- /tts - текст в войс
- /help - помощь
- /text - написать текстовое сообщение
- /voice - записать войс
- /debug - режим отладки
